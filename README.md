# Manual de buenas prácticas para trabajos universitarios en grupo

Repositorio del manual de buenas prácticas para la realización de trabajos universitarios en equipo, gestionado con Git y GitHub.

## Integrantes

- Daniel Cabrita – Ingeniero en Computación
- Marco Rivero – Ingeniero en Computación

## Objetivo del proyecto

Fomentar el uso de Git y GitHub como herramienta de colaboración y control de versiones en la creación de documentos (sin programar), promoviendo buenas prácticas de trabajo en equipo, organización y documentación.
