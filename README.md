## Descripción del proyecto

Este repositorio contiene el manual final del proyecto **"Guía de buenas prácticas para trabajos universitarios en grupo con Git y GitHub"**, elaborado como parte de un trabajo académico de la Universidad Valle del Momboy.

El manual reúne recomendaciones prácticas para mejorar la organización del equipo, la gestión del tiempo, el uso de herramientas de colaboración y el trabajo con Git y GitHub en proyectos universitarios. Además, incluye una breve referencia a las normas institucionales de presentación y citación, con el objetivo de mantener una estructura académica clara y coherente.

Aunque Git y GitHub suelen asociarse con programación, este proyecto demuestra que también pueden utilizarse de forma eficiente en la creación de documentos, informes y manuales colaborativos.

## Contenido del manual

El manual está estructurado en las siguientes secciones:

1. Introducción.
2. Organización del equipo.
3. Gestión del tiempo.

## Flujo de trabajo utilizado

Para desarrollar este proyecto se utilizó un flujo de trabajo colaborativo basado en Git y GitHub:

1. Se definió la estructura del manual.
2. Se dividieron las secciones del contenido.
3. Cada integrante trabajó en una rama propia.
4. Se registraron avances mediante commits descriptivos.
5. Se revisaron los cambios mediante Pull Requests.
6. Se organizaron tareas usando issues.
7. Se etiquetó la versión final del documento.
8. Se generó la entrega final del manual.

Este flujo permitió mantener un historial claro del desarrollo del proyecto y facilitar la revisión entre los integrantes.

## Cómo contribuir

Aunque este proyecto ya fue terminado como entrega académica, el flujo de colaboración que se usó fue el siguiente:

1. Crear una rama de trabajo.
2. Realizar cambios en la sección asignada.
3. Confirmar los cambios con commit.
4. Subir la rama al repositorio remoto.
5. Crear un Pull Request.
6. Revisar los comentarios del compañero.
7. Fusionar los cambios cuando estaban correctos.

Si deseas hacer mejoras personales para uso académico, puedes seguir un proceso similar.
